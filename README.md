## 中文

 Ecotopia 是野生动物 Debut官方APP

## 功能特点

 - 支持金典蓝牙、低功耗蓝牙Intelink功能
 - 支持终端设备OTA功能
 - 地图采用Google 地图和百度 地图.
 - 支持sensor数据图表化分析
 - 支持电子围栏功能
